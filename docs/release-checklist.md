## 緊急時の対応

### 重大なバグが発見された場合

1. すぐに新しいビルドを作成
2. バージョン番号を更新
3. 緊急アップデートをリリース
4. ユーザーに通知（可能な場合）

### 広告の問題

1. AdMobダッシュボードで確認
2. 広告ユニットIDを確認
3. 必要に応じて広告を一時的に無効化

### レビュー却下

1. 却下理由を確認
2. 必要な修正を行う
3. 修正版を再提出
